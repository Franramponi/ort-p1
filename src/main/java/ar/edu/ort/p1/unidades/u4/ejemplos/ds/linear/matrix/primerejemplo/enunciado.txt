Ejercicio 1. Declaracion, definicion y uso de la clase Matriz.

1. Escribir una clase llamada Matriz que implemente las estructuras de datos y
metodos necesarios para manejar matrices cuadradas de numeros enteros, cuya
capacidad puede definir el usuario. Debe incluir los siguientes metodos:
	1.1.	void asignarDatos(int[] vector), rellena el contenido de la matriz
			por filas con los datos pasados desde un vector unidimensional.
	1.2.	void suma(Matriz m), suma los valores de la propia clase con los
			valores de una matriz pasada como argumento.
	1.3.	void producto(Matriz m, multiplica los valores de la propia clase
			con los valores de una matriz pasada como argumento.
	1.4.	void traspuesta(), calcula la traspuesta de la matriz.
	1.5.	void mostrar(), muestra el contenido de la matriz.

2. Escribir un programa llamado Main.java, que realice las siguientes
operaciones:
	2.1. 	Crear dos matrices de tamanio 3x3.
	2.2. 	Rellenar la primera con los datos: 3, 2, 1, 1, 2, 3, 2, 3, 1.
	2.3. 	Rellenar la segunda con los datos: 1, 1, 2, 2, 1, 1, 1, 2, 1.
	2.4. 	Mostrar el contenido de ambas matrices.
	2.5. 	Multiplicar la primera por la segunda y mostrar el resultado.
	2.6. 	Sumar la primera por la segunda y mostrar el resultado.
	2.7. 	Calcular la traspuesta de la primera matriz y mostrar el resultado.
